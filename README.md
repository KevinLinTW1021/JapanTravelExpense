# Travel Expense (旅遊記帳 App)

一個為旅行設計的 iOS 記帳應用程式，支援多種貨幣轉換功能。
往後會新增其他國家幣值的支援。

## 主要功能

- 旅行預算設定
- 支出記錄與分類
- 匯率轉換
- 支出統計與分析

## 技術規格

- iOS 18.0+
- Swift 5.0+
- SwiftUI
- SwiftData 用於數據存儲

## 安裝需求

- Xcode 15.0+
- iOS 18.0+ 設備或模擬器
